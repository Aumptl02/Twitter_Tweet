# Setup

Install Firefox

Install the required dependencies by running:
`pip3 install -r requirements.txt`
or
if using pipenv virtualenv then:
`pipenv install`

add a `.env` file with your username and password in it the structure of it is in `.env.example` file

now you are good to go

# Running

Execute the `twitter_tweet.py` file
You can change the Message of `post_tweet()` method to anything you wish
Same for `like_tweet()` you can change the link of the post
